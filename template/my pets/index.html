花音

12歳　メス　毛色：サビトラ

ツン９割デレ１割気高き女王様

introduction

母の仕事先の方が犬小屋に置き去りにされていたところを保護。当時すでに猫を飼っていたため、我が家で保護することに。
保護当時は目ヤニで目が潰れていましたが無事に回復。現在はきれいなグリーンアイをしています。
先住猫ともすぐに打ち解け、三年後保護した子猫のお世話をするほどほかの猫に優しい性格です。
しかし、人間には気高い一面を見せ、なかなか撫でさせてくれず突然嚙まれたり引っかかれたりすることもしばしば。
ですが我が家ではそれを個性として受け入れ適度な距離感で接しています。

レオ

9歳　オス　毛色：白キジ

甘えん坊の俺様気質

introduction

母が田んぼ道を車で走行していたところ、あぜ道で一匹でいた所を保護。当時三匹の猫がいましたがうちの子として迎え入れることになりました。
保護当時は食事を摂れていなかったのかガリガリでしたが、現在はわがままボディに。
花音にお世話されすくすく育っていましたが反抗期に入ったのか次第にほかの猫に少し気難しい一面を見せるようになりました。
しかし、人間にはとても甘えん坊で寝ているときに肉球を揉んだり、歯を見せてもらっても怒りません。（吸うと嫌な顔をされますが…）
最近は新しく迎え入れた子猫と仲良くできるよう、少しずつトレーニングをしています。

蘭丸

3か月　オス　毛色：白黒

隻眼の超絶やんちゃ坊主

introduction

私の友人が駐車場にいたところを保護。友人宅では飼えないとのことで我が家で引き取ることになりました。 保護当時、花音と同じく目ヤニで目が潰れており点眼薬で治療をしていましたが右目は完治せず腫れてしまったため摘出することに。
右目を失くすハンデを負いましたが今では部屋中を走り回るほど元気になりました。
やんちゃで急に飛びついてきたりもしますが人間が大好きで眠い時はくっついて寝てくれます。
同じ部屋で過ごしているフェレットのブランと姉弟のように遊んでいます。

ブラン

1歳　メス　毛色：セーブル

胴長短足のおっとり眠り姫

introduction

ペットショップで出会い一目ぼれし、先住猫がいるため私の部屋で飼うことに。
まだ一般的には珍しい動物のフェレットで、イタチ科に属しています。胴長短足が特徴で、一日に20時間ほど眠ります。
起きているときは走り回ったり、おもちゃを隠して遊んだり、高いところによじ登ったりと活発な一面も見られます
基本的にはおっとりとしていて私のベッドの上でゴロゴロしたり、おなかを撫でられてもされるがままになっています。
最近同居を始めた蘭丸と遊んであげていたり、種族は違いますがお姉ちゃんのようになっています。
